## SQL Queries

This folder contains SQL queries for analyzing NFL play-by-play data. These queries are designed to extract insights and perform specific analyses.

### Contents
- **last_4_wks_def.sql**: Query for defensive stats over the last 4 weeks.
- **last_4_wks_off.sql**: Query for offensive stats over the last 4 weeks.

Use these queries as templates or modify them for your specific needs.
